# java_springboot_demo
## Project Name: disease-analysis-app
## Key Features:

- Ingest JSON disease records via REST
- Store in PostgreSQL with JPA
- Stream-based analysis (e.g., average severity, filtering)
- Async processing with @Async + CompletableFuture
- Circuit Breaker with Resilience4j
- Security: JWT OAuth2 Resource Server + role-based
- Transaction examples
- N+1 solutions demonstrated
